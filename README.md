# Encontrar somas
Problema 2-sum. Recebemos um grande conjunto A de inteiros e algumas consutas (k) e, para cada consulta, devemos verificar se existe um par de elementos que somados resultem em k.

## Busca Sequencial
Nessa versão, utilizaremos a busca sequencial para verificar se existe um par de elementos do conjuntos que satisfaçam a condição.
